# HopIn

HopIn is a ride-sharing platform for riders and drivers. This folder now contains the real application starter built with simple HTML, CSS, JavaScript, Bootstrap, jQuery, Node.js, Express.js, and Supabase.

## Stack

- Frontend: HTML, Bootstrap, jQuery, custom CSS
- Backend: Node.js, Express.js
- Database and auth: Supabase

## Project direction

The existing root `HopIn/` HTML files are kept as the original prototype reference.
The production-ready app now lives inside `HopIn/Project-App/`.

## Getting started

1. Open the `Project-App` folder
2. Copy `.env.example` to `.env`
3. Add your Supabase credentials
4. Install packages with `npm.cmd install`
5. Start the app with `npm.cmd run dev`
6. Open `http://localhost:3000`

Note: on this Windows machine, `npm.ps1` is blocked by PowerShell execution policy, so use `npm.cmd`.
